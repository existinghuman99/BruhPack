# BruhPack
A rescouce pack full of memes and inside jokes

==============================================
= THIS PACK IS DESIGNED FOR BEDROCK 1.12.201 =
==============================================

It should still work with newer versions, but it has not been tested.
Java has not been tested either, so don't be sad if it does not work.

==============================
= HOW TO IMPORT/RUN THE PACK =
==============================
       
1) Download the pack from the link

2) Go to your device's file explorer and export it to minecraft (hold and select share on IOS, right click and select Open In on PC)

3) Wait for it to import

4) Select the pack in Global Rescources and put it ABOVE the default vanilla minecraft pack

5) If everything went correctly, you should be able to play with it.

==========================
= THINGS TO KEEP IN MIND =
==========================

-This might not work with other packs or shaders

-Playing this pack on Java is not tested and is unstable

-This pack is open source, edit it however you want.
